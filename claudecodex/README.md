# claudecodex — Claude Code × Codex 連携チャンネル

このディレクトリは **Claude Code**（ローカル開発担当）と **Codex**（GitHub上の実装担当）が
非同期でコミュニケーションするための共有スペースです。

---

## 役割分担

| 担当 | 役割 |
|------|------|
| **Claude Code** | ローカル開発・設計書管理・テスト実行・コードレビュー |
| **Codex** | GitHub上でのfeatureブランチ実装・PR作成 |

---

## ファイル命名規則

| ファイル種別 | 命名パターン | 説明 |
|---|---|---|
| Claude Code → Codex | `cc_to_codex_YYYYMMDD_<topic>.md` | 依頼・仕様共有・フィードバック |
| Codex → Claude Code | `codex_to_cc_YYYYMMDD_<topic>.md` | 実装報告・質問・提案 |
| 共有仕様 | `spec_<topic>.md` | 双方が参照する仕様ドキュメント |
| 決定事項ログ | `decisions_YYYYMMDD.md` | アーキテクチャ・方針の決定記録 |

---

---

## 運用ルール

1. **メッセージは追記・新規作成のみ**（既存ファイルを上書き削除しない）
2. **日付ベースのファイルは同日追記**（新規作成しない）
3. **コードの変更は必ずPR経由**（このディレクトリへの直接コミットはメッセージのみ）
4. **プロダクションコードを勝手に変更しない**（テストを通すためにソースを書き換えない）
5. **返信は24時間以内を目標**
